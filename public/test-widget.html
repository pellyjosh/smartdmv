This widget will appear as a floating button on the right side. Look for a floating button on the bottom-right corner.

Widget Features & Position Options

  • ✅ Fully Dynamic: All settings fetched from API (no hardcoded values)
  • ✅ Auto-Updates: Changes in admin panel automatically reflect in widget
  • ✅ Minimal Embed Code: Only practiceId, apiKey, and baseUrl needed
  • ✅ CORS Support: Works on external websites
  • ✅ Position Options: Inline, floating-right, floating-left, modal
  • ✅ Searchable appointment type dropdown
  • ✅ Dynamic loading of practice data
  • ✅ Live preview in admin interface
  • ✅ Embed code display
  • ✅ Error handling for empty states
  • ✅ API key authentication

How the Dynamic System Works:

  • Step 1: Admin configures settings in the dashboard
  • Step 2: Settings are saved to database
  • Step 3: Widget fetches latest settings from API
  • Step 4: Widget renders with current settings
  • Result: No need to update embed code when settings change!
